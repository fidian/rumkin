You can load a GPX or LOC file with waypoints. To make the upload happen faster, you can compress the file in zip or gz formats.

You can use keywords to customize a search. Try "whois:rumkin.com" to look up a whois search on my server. You don't need the "whois:" part typically since many hostnames are detected automatically.

This page also takes parameters. If you append "find=" arguments to the URL, it will automatically search for specific things. To see what I mean, check out 90210 or the multiple find example. If you click on those, check out the URL to see how it all works.
