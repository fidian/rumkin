Übchi

A double columnar transposition cipher that uses the same key, but adds a number of pad characters. Used by the Germans in World War I.

During World War I, the Germans used a double columnar transposition cipher called Übchi ("ubchi" with umlauts). For a bit more information about columnar transposition ciphers, see that cipher's page. This method is surprisingly similar to the U.S. Army's double columnar transposition, also used during World War I.
