Prestige Classes

  • Arcanist - A spellcaster that is able to work with any kind of magic, but trades everything else in their pursuit for these powers.

  • Artificer - A magic user that excels at creating magical items and constructs.

  • Blade Master - A wandering warrior, always searching for better techniques and sharp weapons.

Build Your Own

There is a class construction kit that could be used as a guide to help you make your own 3rd Edition prestige classes.
